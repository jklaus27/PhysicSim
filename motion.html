Motion Lab: Position & Speed

What is motion?

Motion describes how an object’s position changes over time. In this lab, we will watch an object move along a straight line and explore what makes it move faster or slower.

Try it: Constant speed

Use the slider to choose a speed, then press play. Watch how the position changes as time passes.

Think about it

What happens to the position when the speed is zero?
What happens if the speed is negative?
