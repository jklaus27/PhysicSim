Motion Lab: Position & Speed

What is motion?

Motion describes how an object’s position changes over time. In this lab, you will observe a single object moving along a straight path and explore how speed affects its motion.

Explore constant speed

Choose a speed using the slider. Positive values move the object to the right, and negative values move it to the left. Then press play.

Think about it

  • What happens to the object’s position when the speed is zero?
  • How does the direction of motion change when the speed is negative?
  • What do you notice when the object reaches the ends of the track?
